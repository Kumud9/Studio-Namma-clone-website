# Studio Namma Landing Page Clone

A modern, responsive landing page inspired by the Studio Namma website. This project focuses on creating an engaging user experience using HTML, CSS, and JavaScript with smooth animations, interactive navigation, dark mode, and a full-screen video preloader.

## 🚀 Features

* Responsive and modern UI
* Full-screen video preloader
* Animated navigation menu
* Smooth hover effects
* Dark Mode / Light Mode switch
* Interactive hero section
* Custom page transitions
* Clean and organized code structure
* Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Locomotive Scroll
* GSAP (if used)

## ⚡ Getting Started

1. Clone the repository

git clone https://github.com/Kumud9/Studio-Namma-clone-website

2. Navigate to the project folder


cd studio-namma-clone


3. Open `index.html` in your browser.

No additional setup is required.
